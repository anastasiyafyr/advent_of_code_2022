# Advent of Code 2022 (Ruby)

My solutions for [Advent of Code 2022](https://adventofcode.com/2022), written in Ruby.

## Structure

Solutions are organized by day:

- `d1/` — Day 1
- `d2/` — Day 2
- ...

Each day may contain `part_1.rb`, `part_2.rb`, and other helper files.

## Input files

Each puzzle input is unique per Advent of Code account.

This repository does **not** include my personal `input.txt` files.
